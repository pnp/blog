### Demos

* **Create an Azure Communication Services Resource** ‚Äì part 2 in series. Create an ACS resource, then select CallComposute template (from ACS UI Library) to Join Existing (Teams) Call. Set up Teams call, paste meeting link into template. In Azure, generate a User Access token and Identity and paste into template. Start call ‚Äì make camera and sound selections. Try this no code (controls abstract the API surface) solution. Next week ‚Äì custom React apps.
* **What's next for the Microsoft Graph Toolkit!** ‚Äì a show and tell session - what‚Äôs new for v3.0? Refreshed components with Fluent UI, multi-account support, generic picker control, disambiguation built-in, support for Graph JS SDK v3.0, support for sovereign clouds, support for node 14-16. Then what? Conversation component in collaboration with the ACS UI Library - scope is 1:1 and 1:m conversations. Finally, MGT v3.1 and beyond ‚Äì enhanced File List experience and Search Component.
