## Resources related to this blog's content

**Excel**

* [Synchronous custom functions (preview)](https://learn.microsoft.com/office/dev/add-ins/excel/custom-functions-synchronous)
  Preview how to enable synchronous custom functions to run in tandem with certain Excel processes.

* [Use custom functions with LAMBDA functions](https://learn.microsoft.com/office/dev/add-ins/excel/custom-functions-lambdas)
  Learn how to use custom functions as LAMBDA function parameters.

* [Coauthoring in Excel add-ins](https://learn.microsoft.com/office/dev/add-ins/excel/co-authoring-in-excel-add-ins)
  Familiarize yourself with updated guidance for handling coauthoring challenges with Excel add-ins.

* [Script Lab sample: Invocation address sample](https://github.com/OfficeDev/office-js-snippets/blob/main/samples/excel/16-custom-functions/invocation-address.yaml)
  Run the sample in [Script Lab](https://learn.microsoft.com/office/dev/add-ins/overview/explore-with-script-lab) to learn how to reference the custom function’s cell location to adjust behavior.

**PowerPoint**

* [PowerPoint JavaScript API 1.10](https://learn.microsoft.com/javascript/api/requirement-sets/powerpoint/powerpoint-api-1-10-requirement-set?view=powerpoint-js-1.10)
  Try out PowerPoint JavaScript API 1.10 to better manage accessibility, slide background, and hyperlinks in a presentation.  
  [https://learn.microsoft.com/javascript/api/requirement-sets/powerpoint/powerpoint-api-1-10-requirement-set?view=powerpoint-js-1.10](https://learn.microsoft.com/javascript/api/requirement-sets/powerpoint/powerpoint-api-1-10-requirement-set?view=powerpoint-js-1.10)

**Outlook**

* [Identify inline images in the HTML body of a mail item](https://learn.microsoft.com/office/dev/add-ins/outlook/add-and-remove-attachments-to-an-item-in-a-compose-form)
  Learn how to easily locate an inline image in the HTML body of a mail item using the contentId property.  
  [https://learn.microsoft.com/office/dev/add-ins/outlook/add-and-remove-attachments-to-an-item-in-a-compose-form]()

* [OnMessageFromChanged and OnSensitivityLabelChanged events now supported in the unified manifest for Microsoft 365](https://learn.microsoft.com/office/dev/add-ins/develop/event-based-activation#outlook-events)
  Try out the OnMessageFromChanged and OnSensitivityLabelChanged events in an add-in that uses a unified manifest in Outlook on the web and the new Outlook on Windows.  

* [Drag and drop: New properties to determine mouse pointer](https://learn.microsoft.com/office/dev/add-ins/outlook/drag-drop-items)
  Use the taskPaneX and taskPaneY properties to determine the mouse pointer relative to the task pane when dragging and dropping items in Outlook on the web and the new Outlook on Windows.​

## Q&A (Question & Answers)

**Are there any updates regarding the Outlook outage remediation roadmap?**

Jonah Karpman (Product Manager) addressed outages on the previous December community call. Please see [Microsoft outage update](https://youtu.be/idHCs3VUjw0?list=PLR9nK3mnD-OVExAWBr2QtS_5UmqUr7HL4&t=107).

Additionally we’ll do a follow up with more information on the next community call.

**Can we use the drag and drop feature to drag and drop from new Outlook to an external app?**

Currently, we just support drag and drop into the task pane.

**Are there any drag and drop features planned for apps other than Outlook?**

Just Outlook at this time.
